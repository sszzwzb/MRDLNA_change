# Dependencies

- iOS DLNA Function 
- iOS DLNA 投屏功能, 支持各大主流电视盒子(小米,华为,乐视,移动魔百盒等), 可以播放,暂停,快进退,调音量,退出.

## 改变
#### 添加进度条回调，添加结束回调等

# Usage

```
原地址和版本
pod 'MRDLNA' ,  '~> 0.1.1'

添加功能后的地址
pod 'MRDLNA', :git => 'https://github.com/sszzwzb/MRDLNA_change.git'
```
